Quest Log

DevOps Engineer at Capgemini: HPE

During my time at Capgemini, I had the opportunity to work with HPE on the development of a Data Fabric project. This was a global data infrastructure designed for storage and management, aimed at providing efficient and scalable solutions for data storage challenges. To manage and deploy MapR and Spectrum Scale clusters in a containerized manner, I utilized Helm Charts and Docker. These tools enabled me to streamline deployment processes and maintain consistency across different environments. I was responsible for the entire CI/CD process of our application. This involved pulling the code from my local machine to version control software like Git, and using Jenkins to build the code. By managing the entire pipeline, I ensured seamless transitions between development, testing, and production stages. As part of my responsibilities, I managed the end-to-end workflow of the system using Kubernetes. This allowed me to orchestrate and automate container deployment, scaling, and management, ensuring high availability and performance across the application. To further enhance our deployment process, I used ArgoCD to monitor the Git repositories storing our Kubernetes manifests. This allowed for automatic deployment of any changes to the Kubernetes clusters, ensuring that our infrastructure remained up-to-date and secure. In addition to managing deployments, I also focused on testing and quality assurance. I automated more than 60 test cases using Python and the ROBOT Framework, while also creating and executing over 200 manual test cases. This helped me identify potential issues and ensure the reliability of our application. Throughout my work on the project, I used Git to manage repositories, merge code, and resolve conflicts. This ensured that our team could collaborate effectively and maintain a streamlined and efficient development process.

Java Developer at Marlabs Inc.

In one of my projects at Marlabs Inc., I was involved in the development of a web-based stock market system. The primary goal of this system was to provide users with real-time stock prices and help them make informed investment decisions. To build a scalable and efficient application, I utilized microservices architecture based on Spring Boot and Spring Cloud. This approach allowed me to create independent, modular components that could be easily maintained and updated. To manage logging within the application, I used Spring AOP, which enabled me to monitor the application status and gather important information for debugging and optimization purposes. Security was a top priority for the stock market system, given the sensitive nature of the data it handled. To ensure the highest level of protection, I implemented Spring Security with OAuth2.0 protocol. This provided a robust and reliable authentication and authorization framework, safeguarding user data and ensuring secure access to system resources.

Game Development Journey

As a passionate game developer, I've had the opportunity to create several exciting games using a range of programming languages and game engines. My journey began with developing classic games such as 'Pong', 'Space Invaders', and 'Tetris' using Python. This experience allowed me to hone my skills in developing simple yet engaging gameplay mechanics and understanding the nuances of game logic and player interaction.

I then progressed to more complex game development with 'Terminal Hacker', a game I coded in C# and developed using Unity. This project was particularly challenging and rewarding, as it involved more advanced programming concepts and allowed me to explore Unity's powerful features for creating immersive game environments and interactive gameplay.

One of my most ambitious projects has been developing 'Warehouse Wreck' in Unreal Engine 5 using C++. In this game, I implemented complex physics to the actors, which significantly enhanced the realism and interactivity of the game world. This experience deepened my understanding of physics simulation in games and the capabilities of Unreal Engine in rendering high-quality graphics and dynamic environments.

Throughout my game development career, I've also focused on incorporating advanced concepts such as AI pathfinding, networked multiplayer functionality, and procedural content generation. These elements have allowed me to create more sophisticated and engaging gaming experiences. My journey in game development continues to be a source of immense learning and creative fulfillment.
